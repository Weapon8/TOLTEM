# TolTem — Calculateurs

Calculateurs tôlerie autonomes (HTML), hébergés via GitHub Pages.

- **Trop-Plein** — développé à plat (U, capot, pente, bec)
- **EP / Fourreaux** — zinc/galva et fourreaux (table + formule)
- **Boîte à eaux (BAE)** — corps, trop-plein, EP (position perso), sortie arrière · vue 3D + développé
- **Fourreau 1.5** — fourreau carré/rectangle · 4 pièces (haut/bas/gauche/droite), pli platine, angles de pliage · vue 3D + développé
